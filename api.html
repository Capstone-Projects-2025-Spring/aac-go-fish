This API controls basic lobby functionality of the AAC game

Create Lobby

Creates a new lobby, returning lobby code.

Responses

https://capstone-projects-2025-spring.github.io/lobby

Response samples

  • 201
Content type
application/json
{
  • "code": "Fish-Turtle-Shark"
}

Join Lobby

Joins a player to a lobby by its unique game code, returning new player information and player avatars.

path Parameters
code
required
string (Code)

Lobby Code

Responses

https://capstone-projects-2025-spring.github.io/lobby/{code}

Response samples

  • 200
Content type
application/json
{
  • "self": {
    • "avatar": "Fish",
    • "uuid": "3e6b52bf-23ae-4f50-ba84-ad35ed1df31b"
    },
  • "players": [
    • "Shark",
    • "Turtle",
    • "Fish"
    ]
}
